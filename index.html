ประเภทของเครื่องแม่ข่าย (Server)

รู้จักกับเครื่องแม่ข่ายแต่ละประเภท พร้อมซอฟต์แวร์ที่นิยมใช้งาน

1. เครื่องแม่ข่ายเว็บ (Web Server)

หน้าที่หลัก: ให้บริการเนื้อหาเว็บไซต์แก่ผู้ใช้งานผ่าน HTTP/HTTPS

ตัวอย่างซอฟต์แวร์:

  • Apache HTTP Server
  • Nginx
  • Microsoft IIS

2. เครื่องแม่ข่ายฐานข้อมูล (Database Server)

หน้าที่หลัก: จัดเก็บและให้บริการข้อมูลผ่านระบบฐานข้อมูล

ตัวอย่างซอฟต์แวร์:

  • MySQL / MariaDB
  • PostgreSQL
  • Microsoft SQL Server
  • Oracle Database

3. เครื่องแม่ข่ายไฟล์ (File Server)

หน้าที่หลัก: ให้บริการแชร์ไฟล์ภายในเครือข่าย

ตัวอย่างซอฟต์แวร์:

  • Samba
  • Windows Server File Services
  • NFS (Network File System)
